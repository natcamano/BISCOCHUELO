# Paso 3 · Batir yemas con azúcar

Este paso desarrolla la estructura base del biscochuelo y aporta esponjosidad.

## Instrucciones

1. Agregar el azúcar sobre las yemas.
2. Batir con batidora eléctrica a velocidad media-alta durante **5 minutos aproximadamente**.
3. La mezcla está lista cuando triplica su volumen, se vuelve de color amarillo pálido y cae en cinta al levantar el batidor.

## Por qué importa

Al batir se incorpora aire a las yemas y el azúcar se disuelve completamente. Esta base aireada es lo que sostiene el volumen final del biscochuelo.

## Tips

- No apurar este paso: los 5 minutos son importantes.
- Si no tenés batidora eléctrica se puede hacer a mano, pero requiere más tiempo y esfuerzo.

## Checklist

- [ ] Mezcla de color amarillo pálido
- [ ] Volumen visiblemente aumentado (aprox. el triple)
- [ ] Cae en cinta continua al levantar el batidor

# 🍰 Biscochuelo Clásico

Guía paso a paso para preparar un biscochuelo esponjoso y perfecto.

## Estructura del repositorio

```
biscochuelo/
├── 01-preparacion/
├── 02-separar-huevos/
├── 03-batir-yemas/
├── 04-claras-punto-nieve/
├── 05-incorporar-harina/
├── 06-unir-claras/
├── 07-hornear/
└── 08-desmoldar/
```

## Ingredientes base (molde de 24 cm)

| Ingrediente | Cantidad |
|---|---|
| Huevos | 4 |
| Azúcar | 150 g |
| Harina 0000 | 150 g |
| Sal | 1 pizca |

## Pasos rápidos

1. [Preparación previa](./01-preparacion/README.md)

## Tiempo total

- **Preparación:** 20 min
- **Horneado:** 30–40 min
- **Enfriado:** 20 min mínimo
